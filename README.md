# TurboTaxi
> Simple text-based game created in Java using Lanterna library.

![](https://github.com/kabix-dk/TurboTaxi/blob/master/images/1.1.jpg)

## General Info
Game was made as a project for studies. It uses MVC Design Pattern.

## Technologies
* Java
* Lanterna - 3.0.1
